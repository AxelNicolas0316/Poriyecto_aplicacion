## 🎨 Sistema de Diseño

### Paleta de Colores

| Color | Hex | Uso |
|-------|-----|-----|
| Primario | `#4f46e5` | Botones, enlaces, acentos |
| Secundario | `#06b6d4` | Elementos secundarios |
| Terciario | `#8b5cf6` | Gradientes, detalles |
| Éxito | `#10b981` | Mensajes positivos |
| Advertencia | `#f59e0b` | Alertas |
| Peligro | `#ef4444` | Errores |

### Gradientes Exclusivos

```css
Primary:   Índigo → Violeta
Secondary: Cian → Índigo
Success:   Verde → Cian
Warm:      Ámbar → Rojo
Cool:      Índigo → Cian
```
