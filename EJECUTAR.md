## 🐛 PROBLEMAS COMUNES

### ❌ "npm no se reconoce"
**SOLUCIÓN:** Instala Node.js desde https://nodejs.org

### ❌ "Puerto 5173 en uso"
**SOLUCIÓN:**
```bash
taskkill /F /IM node.exe
npm run dev
```

### ❌ Pantalla blanca/negra
**SOLUCIÓN:** 
1. Presiona F12 en el navegador
2. Ve a la consola
3. Dime qué error rojo aparece

### ❌ "Cannot find module"
**SOLUCIÓN:**
```bash
npm install
npm run dev
```
